# Karaoke Reservation System

A desktop-based reservation system built for managing karaoke room bookings, customer transactions, and operational reports.

## Features
- **Booking Management**: Check room availability and handle new reservations.
- **Cashier Module**: Process payments and manage transaction history.
- **Multi-Role Access**: Dedicated access for Admin, Cashier, and Manager.

## Tech Stack
- **Language**: Java
- **UI/UX**: Java Swing
- **Database**: MySQL

## Role
**Full-Stack Desktop Developer (Cashier Module)**
Responsible for the end-to-end development of the cashier module, including UI implementation, application logic, and database integration.

## How to Run
1. Clone this repository.
2. Import the project into your IDE (Eclipse/IntelliJ).
3. Configure the database connection in the project settings.
4. Run the main class.
